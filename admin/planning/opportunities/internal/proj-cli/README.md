## ðŸŽ¯ Overview

proj-cli is a unified CLI tool that provides:
- Project management commands (create, list, get, update, delete)
- Inventory scanning and analysis
- Template-based project creation (ported from dev-infra)
- Local registry for tracking template-created projects
- Project type filtering (Work, Personal, Learning, Inactive)

**Project Context:**

- **Type:** CLI Tool
- **Technology Stack:** Python, Typer, Rich, Pydantic
- **Purpose:** Unified interface for project and inventory management
- **Status:** Active development (v0.2.0 released, Project Type Support in progress)
