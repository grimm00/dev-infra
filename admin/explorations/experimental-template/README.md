## ðŸŽ¯ Overview

Based on ADR-003 (Command Tiers), explore creating an "experimental template" that includes commands not yet stable enough for the standard template.

**Concept:**
- Standard template: Essential + Valuable + Advanced (stable)
- Experimental template: Standard + Evolving (being tested)

This allows users who want cutting-edge features to opt-in while keeping the standard template stable.
